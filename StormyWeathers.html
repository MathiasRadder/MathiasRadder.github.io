Stormy Weathers

Stormy Weathers itch We created this game during a game jam in a group. It is a flying game where you have to fly in calm and stormy areas and try to survive as long as possible.

The game

Stormy Weathers itch

This was a group project with four people, more info on the itch page about it! It a fun time and good teamwork that made this game. We only had four days to create the game from scratch. The game is made in unity.

Our keywords for the game was peace and chaos and so we created a chaotic flying game with some peaceful intervals.

For only four days our game was a bit ambiguous, but we still learned a lot from it! I learned a lot how to work in Unity and creating the planes movement.

What i learned and did

- I created the movement and physics of the plane.

- Made sure that the game loop was in order.

- Creating the mechanics of the UFO and Flying Bomber.

- Replacing simple animation with code.

- Working well together with teammates in a short amount of time.

- Created part of the lighting.

- Helping my fellow programmer and she also helping me with parts.
